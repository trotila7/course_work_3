# Приложение для анализа транзакций

Это приложение позволяет анализировать банковские транзакции из Excel-файла, генерировать JSON-данные для веб-страниц, а также формировать Excel-отчеты.

## Установка

Для начала клонируйте репозиторий

```bash
git clone https://github.com/trotila7/course_work_3
```
Установите все зависимости:

```bash
poetry install
```
